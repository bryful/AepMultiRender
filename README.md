# AepMultiRender
After Effectsのレンダリング用コマンドaerender.exeをバックグラウンドで複数起動させるものです。


# Dependency
Visual studio 2017 C#
DynamicJson


# Setup
好きな場所に置いてください。
僕は適当な場所において、デスクトップにショートカット置いて使っています。

# Usage
アプリアイコンにaepファイルをドラッグ&ドロップしてください。
または起動状態なら表示されているフォームにaepをドラッグ&ドロップしてください。

起動するaerenderの数を指定。メモリが32Gなら8個が限界でしょう。
目安としてはPCの搭載メモリを4で割った数で指定すればいいでしょう。
